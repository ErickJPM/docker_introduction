¿Alguna vez has escuchado el término "Contenedor"?

Un contenedor es un espacio de almacenamiento capaz de contener información completamente aislada de otra.Con ayuda de un contenedor un desarrollador es capaz de implementar todos los recursos que necesita para el desarrollo de un proyecto sin la necesidad de que tenga contacto con algún otro proyecto, es decir, toda la información módulos y el resto de elementos necesarios van a ubicarse dentro de un espacio independiente
