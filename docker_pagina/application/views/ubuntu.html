¿Porqué no hablamos un poco acerca de Ubuntu?

Ubuntu es un sistema operativo de código abierto que permite muchas libertades a la hora de desarrollar algún proyecto.Ubuntu permite la instalación de software tal como lo es docker de una manera completamente sencilla y no solamente sino que además es posible utilizar herramientas de desarrollo que en otros sistemas operativos resultaría muy difícil o a veces imposible de utilizar de una forma muy sencilla.
